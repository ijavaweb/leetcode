## 206. Reverse Linked List

* **description**

  Reverse a singly linked list.

  **Example:**

  ```
  Input: 1->2->3->4->5->NULL
  Output: 5->4->3->2->1->NULL
  ```

* **solution**

  ```java
  class Solution {
      public ListNode reverseList(ListNode head) {
          ListNode  prev = null;
          ListNode cur = head;
          while(cur != null){
              ListNode node = new ListNode(cur.val);
              node.next = prev;
              prev = node;;
              cur = cur.next;
              
          }
          return prev;
      }
  }
  ```
